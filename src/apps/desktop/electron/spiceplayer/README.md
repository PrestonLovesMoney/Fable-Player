# Fable Player

Desktop music player for SoundCloud — Electron + React.

## Quick start

```bash
cd src/apps/desktop/electron/fableplayer
npm install
npm run dev
```

Register `fableplayer://callback` as the OAuth redirect URI in your SoundCloud app settings.

Set in `.env`:

```
SOUNDCLOUD_CLIENT_ID=...
SOUNDCLOUD_CLIENT_SECRET=...
```
